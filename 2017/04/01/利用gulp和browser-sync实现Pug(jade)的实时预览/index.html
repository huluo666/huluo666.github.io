一、Pug安装编译

$ npm install -g pug #全局安装

这里如果遇见了

还需要安装pug-cli，具体原因，Github给出的解释是这样的：

cli是命令行端的程序。我们通过命令行来编译pug文件。

pug -h— 查看帮助

pug demo.pug—在同级目录渲染并生成html文件
