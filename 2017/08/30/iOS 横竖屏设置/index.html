iOSжЁӘз«–еұҸи®ҫзҪ®
2017-08-30|iOS

iOSжҢҮе®ҡйЎөйқўй»ҳи®ӨжЁӘеұҸпјҢиҮӘз”ұеҲҮжҚў

  • modelдёӢпјҢpresentж–№ејҸжҺЁеҮәз•ҢйқўгҖӮ

  • pushжЁӘеұҸпјҢеёҰtabbarгҖҒnavigationпјҢдё”дёҖдёӘitemдёӢжүҖжңүжҺ§еҲ¶еҷЁеҜ№еә”зҡ„еҸӘжңүдёҖдёӘж №navigationгҖӮ

1гҖҒдҝ®ж”№е·ҘзЁӢзҡ„info.plistдёӯвҖқSupported interface orientationsвҖқзҡ„еҖјпјҲдёҖиҲ¬еңЁе·ҘзЁӢзҡ„Taget-> General -> Deployment Info -> Device OrientationеӨ„жү“й’©жқҘйҖүжӢ©и®ҫеӨҮж”ҜжҢҒпјү

2гҖҒе®һзҺ°е·ҘзЁӢзҡ„AppDelegateж–Үд»¶дёӯзҡ„пјҲapplication:supportedInterfaceOrientationsForWindow:пјүж–№жі•пјҢеңЁжӯӨж–№жі•дёӯиҝ”еӣһзЁӢеәҸж”ҜжҢҒзҡ„ж–№еҗ‘жһҡдёҫгҖӮ

1гҖҒе…ҲеңЁAppDelegateдёӯйҮҚеҶҷsupportedInterfaceOrientationsForWindowж–№жі•

2гҖҒеҲӣе»әдёҖдёӘUINavigationControllerзҡ„еӯҗзұ»BaseNaviControllerпјҢи®ҫзҪ®self.windowзҡ„RootViewControllerдёәBaseNaviController;

з„¶еҗҺпјҢжҲ‘д»¬еңЁBaseNaviController.mж–Үд»¶йҮҢйқўд№ҹе®һзҺ°зқҖдёӨдёӘж–№жі•пјҡ

3.еңЁз¬¬дёҖдёӘйЎөйқўеҶҷдёҠпјҢж”ҜжҢҒж—ӢиҪ¬ дҪҶжҳҜиҝҷдёӘйЎөйқўеҸӘж”ҜжҢҒз«–еұҸпјҢд»Јз Ғпјҡ

4.з¬¬дәҢдёӘйЎөйқўжҲ‘д»¬еҶҷдёҠпјҢжҲ‘д»¬жүҖйңҖиҰҒзҡ„жЁӘеұҸж•ҲжһңпјҢд»Јз Ғпјҡ

еҪ“еҸӘйңҖиҰҒй”Ғе®ҡеұҸе№•зҡ„ж—ӢиҪ¬дёҚйңҖиҰҒи®ҫзҪ®еұҸе№•зҡ„ж—ӢиҪ¬ж–№еҗ‘ ( ж–№еҗ‘еӣәе®ҡжҳҜз«–зӣҙ)

ејҖеҗҜеұҸе№•ж—ӢиҪ¬е№¶и®ҫзҪ®еұҸе№•ж—ӢиҪ¬ж”ҜжҢҒзҡ„ж–№еҗ‘

еҜ№дәҺйқһmodalжЁЎејҸзҡ„viewController:еҰӮжһңдёҚжҳҜrootViewController,еҲҷйҮҚеҶҷsupportedInterfaceOrientations,preferredInterfaceOrientationForPresentationд»ҘеҸҠshouldAutorotateж–№жі•, жҢүз…§еҪ“еүҚviewControllerзҡ„йңҖиҰҒиҝ”еӣһе“Қеә”зҡ„еҖј.еҰӮжһңжҳҜrootViewController,еҲҷеҰӮдёӢйҮҚеҶҷж–№жі•:

жҳҫиҖҢжҳ“и§Ғ, жҲ‘д»¬е·§еҰҷзҡ„з»•ејҖдәҶUIKitеҸӘи°ғз”ЁrootViewControllerзҡ„ж–№жі•зҡ„и§„еҲҷ. жҠҠеҶіе®ҡжқғдәӨз»ҷдәҶеҪ“еүҚжӯЈеңЁжҳҫзӨәзҡ„viewController.
