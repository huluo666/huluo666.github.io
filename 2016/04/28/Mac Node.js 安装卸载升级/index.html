Mac 下安装node.js

http://www.jianshu.com/p/f21fdbdf47df

确保mac电脑上已经安装homeBrew，使用下面命令安装node.js

brew install node

MAC node.js升级

node.js升级步骤

第一步，先查看本机node.js版本：
$ node -v

mac 卸载node

brew 安装的node ，命令卸载

brew uninstall --force node
