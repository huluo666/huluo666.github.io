Mac ffmpeg 使用

1、安装ffmpeg—brew install ffmpeg

如果想要添加libfdk_aac或libvpxd等库(这两个库是高度推荐安装的)，可以输入以下包含额外推荐选项的命令:

ffmpeg需要的依赖包几乎都可以通过brew install安装：

也可以通过一下网址下载

  • http://evermeet.cx/ffmpeg/
  • http://ffmpegmac.net/

2、查看ffmpeg info—brew info ffmpeg

3、升级ffmpeg brew update && brew upgrade ffmpeg

示例

  1. 使用命令ffmpeg -i http://...m3u8 -c copy out.mkv将视频流下载并合并成 out.mkv。
